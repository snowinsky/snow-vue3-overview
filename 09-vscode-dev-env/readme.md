# vscode 开发 vue工程

- 安装nodeJs
- 安装npm
- 安装cnpm
     - npm install -g cnpm --registry=https://registry.npmmirror.com
- 安装vue cli脚手架或者vite脚手架(官方建议用vite，但vite不支持动态url)
    - cnpm install -g @vue/cli
    - $ npm create vite@latest
- 创建一个vue工程
    - vue create projectName1
    - $ npm create vite projectName2
- 安装依赖
    - npm install
    - npm install
- 开发环境运行
    - npm run serve
    - npm run dev
- 打包
    - npm run build
    - npm run build
